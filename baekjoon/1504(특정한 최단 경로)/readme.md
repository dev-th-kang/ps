## 특정한 엣지를 거치는 최단경로는 어떻게 구할까?

* v1과 v2를 통하는 최단거리를 구하려고한다.
  * 1 -> v1 -> v2 -> 마지막vertax
  * 1 -> v2 -> v1 -> 마지막vertax

* 위의 두개의 경로중에서 더 짧은 거리가 정답이 된다.

<hr>

## 저렇게 구하는 이유
1번 vertax에서 v1에 먼저 도달할 수도 또한 v2에 먼저 도달할 수도 있다, 이 2가지 경우의 수를 모두 포함해서 문제를 해결하면 해결할 수 있다. 무조건 한번에 처리할려고하는 생각은 버리는게 좋다.
